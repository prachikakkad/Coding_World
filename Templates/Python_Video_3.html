2. Rules for Naming Variables in Python

There are several rules in python for defining variable names :-

1. Variable names can contain only letters , numbers and underscores.

2. Variable names can start with a letter or an underscore , but not with a number.

3. No Extra spaces are allowed.

4. Don't use Python Keywords and functionnames as variable names. (Will see list of Python Keywords in Upcoming Videos.)

5. Be careful when using the lowercase letter l and the uppercase letter O because they could be confused with the numbers 1 and 0.

6. You should use meaningful variable names in your program. Variable's name should be short but descriptive as well.

5. What are Data Types ?

Data types specify that which type of data (value) is stored in a variable.
For Example :-
Number = 10
This example is of int data type as it is storing a number in it.

6. How many Data Types are there in Python and What are they ?

Primarily , there are 9 Data Types in Python :-

1.Int
2.Float
3.Boolean
4.None
5.String
6. List
7.Tuple
8.Dictionary
9. Set

We will learn about first 4 data types in this video and other data types separately.

1. Int :- It stores a number in it. But not decimal numbers.
2. Float :- It stores floating point number decimal number in it.
3. Boolean :- It stores True or False in it. In python , 0 means False and all other values mean True.
4. None :- It stores None value in it.
5. Strings :- The value stored inside Quotation Marks.
