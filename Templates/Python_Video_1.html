1. What is Programming ?

Right now , I am communicating with you so , I am using a language/medium to express myself. The same way , we use a medium called Programming to communicate with our Hardware/Computer.

2. Programming Languages and its Examples ?

As we now know that we use a medium called Programming to communicate with our Hardware/Computer. So, for that , it is required to learn a Programming Language. Examples are :- Python , HTML , CSS , JavaScript , Java , PHP , C , C++ , Kotlin , etc.

3. Why We Have to Learn Programming Languages ?

I am able to communicate with you because you and me both know Hindi Language. But Computer don't know/understand any of our languages (like Hindi , English , etc.) So , at the present , we can't teach the computer our languages , so we only have to learn Programming Language(s).

But let me tell you , Computer can't understand Programming Languages also. Then why should we learn them ? Computer only understands one language and that is Binary Language. It is very difficult to learn and work with Binary Language. But Computer can convert Programming Languages to Binary Language. So, if we learn Programming Language(s) , and give instructions to Computer using these languages , then Computer can convert those instructions to Binary and work according to those instructions.

So that is the reason we have to learn Programming Languages. I will give more information about Binary Language to you in upcoming videos.
